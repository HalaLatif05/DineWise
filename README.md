# DineWise

A UX/UI design case study proposing a personalized food delivery application that reduces decision fatigue through context-aware recommendations and user-centered design principles.

---

## Project Overview

DineWise is a personalized food delivery application prototype designed to simplify meal selection by recommending food based on the user's context, habits, and time of day.

The project follows the Human-Computer Interaction (HCI) design process, including user research, prototyping, usability testing, and accessibility evaluation to create a more intuitive user experience.

---

## Objectives

- Reduce decision fatigue during meal selection
- Provide personalized meal recommendations
- Improve usability and accessibility
- Design an intuitive mobile interface
- Apply Human-Computer Interaction principles

---

## Features

- Time-based recommendations (Morning & Night modes)
- Personalized meal suggestions
- Smart recommendation popup
- Restaurant browsing interface
- User-centered mobile UI

---

## Tools Used

- Figma
- User Interviews
- Usability Testing
